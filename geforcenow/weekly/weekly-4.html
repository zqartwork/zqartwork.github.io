GFN 週報
將針對《巫師》系列的 GOG 版本提供更多支援服務

2021年03月XX日 發佈

本週包含《要塞攻防戰：十字軍》(Stronghold: Crusaders)等，共九款遊戲加入 GeForce NOW

本週 GFN十分特別，因為我們很高興能分享這項消息，我們的雲端串流服務為GOG加入了更多支援服務，而這是我們持續開放平台策略的一環。
於 GOG 平台上擁有《巫師》(The Witcher) 系列遊戲的會員，現在可跨裝置暢玩不同版本。這項服務採用 GeForce NOW 的串流伺服器，強大的力量連維京人都會嘖嘖稱奇。我們未來也計劃與 GOG 合作舉辦更多活動。

不過首先，讓我們來談談 GOG 和傑洛特


這是屬於你的故事

CD PROJEKT RED 的《巫師》(The Witcher) 系列銷量已超過 5000萬套，世界各地的玩家都已經加入傑洛特的冒險旅程。

此外，GOG 是一個十分重視顧客的數位發行平台，而他們以PC為主要發展的理念也讓我們成為他們的忠實粉絲。他們還打造了GOG GALAXY，這個應用程式可將多個遊戲庫整合成單一遊戲庫，也能讓你與好友跨遊戲平台保持連繫。

自從 CDPR 的《電馭叛客 2077》(Cyberpunk 2077) (GFN 上首款支援 GOG 的遊戲) 發行後，我們一直與 GOG 團隊保持合作，為更多遊戲加入支援。任務完成後，我們與 GOG 的合作夥伴共同進行第二階段工作。

這個階段從今天開始。

在 GOG 上擁有下列遊戲的會員，可直接從我們的雲端遊戲伺服器跨裝置串流遊戲。
多虧有了 GeForce NOW 的分享工具，讓會員可以在《巫師 3：狂獵》(The Witcher 3: Wild Hunt)年度最佳遊戲版本中，輕鬆擷取精彩時刻的遊戲畫面。

你可以使用我們的遊戲內迷你窗，拍下擊倒怪物時的榮耀時刻，或是使用 NVIDIA Freestyle 套用代表造型。

你還可以使用Ansel從任何角度進行構圖，將傑洛特在CDPR 絕美開闊世界中的英姿，變成你的最佳相片。

更多遊戲
會員可以在以下 GeForce NOW遊戲庫 尋找新加入的遊戲，包含《巫師》(The Witcher)系列中的 GOG.com 支援遊戲：

  • 《王座的崩毀：巫師軼聞》（(Thronebreaker: The Witcher Tales） ( GOG.COM )
  • 《巫師：冒險遊戲》（The Witcher Adventure Game） ( GOG.COM )
  • 《巫師 2：王國刺客加強版》（Assassins of Kings Enhanced Edition） ( GOG.COM )
  • 《巫師 3：狂獵》（The Witcher 3: Wild Hunt） ( 年度最佳遊戲版 GOG.COM )
  • 《要塞攻防戰：軍閥之戰》 ( Stronghold: Warlords ) ( 將於 Steam 新發行 )
  • 《Monster Energy Supercross - The Official Videogame 4》( 3 月 11 日於 Steam 同步發行 )
  • 《帕斯卡契約：終極版》( Wager: Definitive Edition )( 3 月 12 日於 Steam 同步發行 )
  • 《Uno》( Steam )
  • 《工人與資源：蘇維埃共和國》( Workers & Resources: Soviet Republic )( Steam )
