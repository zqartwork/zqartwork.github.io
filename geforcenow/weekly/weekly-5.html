GeForce NOW 持續優化會員遊戲體驗

2021年03月XX日 發佈

非常感謝所有會員提供的意見回饋，我們會用於改善未來的服務及規劃。

在提及新的規劃之前，先跟會員們分享去年的改善內容，包括：

  • 支援同步發行 ── 例如《電馭叛客 2077》( Cyberpunk 2077 ) 和熱門免費遊戲，擴增遊戲庫，現在遊戲庫內含 800 多款遊戲。
  • 開放全新的資料中心 ── GeForce NOW 在全球享有超過 20 個資料中心，擴大現有資料中心的容量，讓我們能夠進一步拓展服務，提升服務品質。


即將推出的最佳化功能

2.0.28 版更新將提供給會員使用，讓串流品質提升到全新境界，我們其中一個改善方法就是採用獨特的 Adaptive VSync 技術。

此功能可在伺服器端以 60 或 59.94 Hz 同步畫面播放速率，以配合用戶端的顯示器，減少支援遊戲出現畫面間斷停頓和延遲的情況。全新的調整抖動技術將能讓我們提升位元速率，並改善斷斷續續的網路品質。

我們也竭盡所能讓 GeForce NOW 變得更好用，包括提供讓會員更快進入遊戲的更新。這些改善內容的第一階段包括在未來 1 到 2 個月內，於平台上連結重要遊戲的帳戶，以及提供遊戲預先載入的更新，預計可減少約一半的載入時間。

GeForce NOW 引進全新地區消息

GeForce NOW 不久前在土耳其宣布了全新的合作夥伴，也即將與沙烏地阿拉伯和澳洲合作。

GeForce NOW 聯盟合作夥伴會負責區域資料中心的營運，並提供 GeForce NOW 以當地貨幣支付的產品以及當地語言支援。

如此一來，便能減少延遲、大幅度改善連線速度，並減少等待時間。
我們預期很快會有更多的 GeForce NOW 聯盟合作夥伴加入，拓展至全新地區。


本週遊戲精彩內容

2020 年時，我們每週平均上架 10 款遊戲。我們的目標是運用全新的快速上架流程，也持續增加對其他數位遊戲商店的支援。

如果你還沒聽說，我們在上週推出了四款 CD PROJEKT RED 遊戲的 GOG.COM 版本，包括《巫師 3：狂獵》(The Witcher 3: Wild Hunt) 年度最佳遊戲版。

本週 GFN 週報除了 2.0.28 版服務即將更新的消息之外，我們還會在 GeForce NOW 上串流 7 款新遊戲。

會員們可以期待：

  • 《不要餵猴子》（Do Not Feed the Monkeys） ( Steam )
  • 《GoNNER》( Steam )
  • 《Loop Hero》( Epic Games Store )
  • 《地產大亨Plus》( Monopoly Plus )( Steam )
  • 《Snooker 19》( Steam )
  • 《網路奇兵：加強版》( System Shock: Enhanced Edition )( Steam )
  • 《丸霸無雙》( Wanba Warriors )( Steam )
